1. Introdução
O projeto consiste no desenvolvimento de uma página web informativa sobre IDEs (Integrated Development Environments) que, de forma comparativa, apresenta três dessas ferramentas que são frequentemente utilizadas na área de desenvolvimento de softwares, sendo elas o Visual Studio Code,o Apache NetBeans e o Android Studio.
A aplicação busca oferecer uma experiência intuitiva de navegação, utilizando um menu fixo na parte de cima da página que permite o acesso direto a cada tópico do texto. O objetivo é facilitar a compreensão das principais características de cada uma dessas IDEs, incentivando o aprendizado e o interesse de estudantes e iniciantes na área de programação.

2. Objetivo do Sistema
O sistema tem como objetivo principal disponibilizar um ambiente online leve, responsivo e de fácil utilização, que permita ao usuário explorar, comparar e compreender as características, vantagens e limitações de três IDEs populares: Visual Studio Code, Apache NetBeans e Android Studio.
Além de seu uso informativo, o projeto possui uma proposta educacional, auxiliando estudantes e novos programadores a compreenderem as vantagens, limitações e contextos de uso de cada IDE, o que permite a escolha consciente de ferramentas de desenvolvimento por meio de informações organizadas e de fácil acesso..

3. Público-Alvo
O público-alvo compreende estudantes, iniciantes em programação e desenvolvedores em formação, especialmente aqueles que buscam compreender as diferenças entre IDEs populares e identificar qual delas melhor se adapta a suas necessidades e objetivos de aprendizado.


4. Arquitetura do Sistema
O sistema foi desenvolvido como uma aplicação web estática, estruturada em uma única página (modelo Single Page Application simplificada). O conteúdo é dividido em seções correspondentes a cada IDE, com navegação interna realizada por meio de âncoras e um menu fixo superior. Assim garantindo a leveza, rapidez no carregamento e compatibilidade com diferentes dispositivos, promovendo uma boa experiência de uso.

5. Funcionalidades Principais
Menu fixo no topo com links para seções do conteúdo.
Navegação rápida e intuitiva entre tópicos.
Exibição de texto e imagens explicativas sobre cada IDE.
Layout responsivo para diferentes dispositivos.

6. Levantamento de requisitos
   
6.1. Requisitos funcionais:
RF01 - Menu
O sistema deve exibir um menu no topo com os tópicos: Introdução, Visual Studio Code, Apache NetBeans, Android Studio e Conclusão.
RF02 - Redirecionamento
Ao clicar em um item do menu, o usuário deve ser direcionado para a seção correspondente da página, sem rolar manualmente.
RF03 - Apresentação do texto
O sistema deve exibir informações textuais sobre IDEs (Visual Studio Code, Apache NetBeans e Android Studio).
RF04 - Logos ou imagens
O sistema deve exibir imagens/logos relacionadas a cada IDE em sua respectiva seção.
RF05 - Organização
O sistema deve organizar cada o texto de forma que cada um dos tópicos esteja devidamente separado

6.2. Requisitos não funcionais:
RNF01 - Tempo de carregamento
O tempo de carregamento da página não deve ultrapassar 3 segundos em conexões comuns.
RNF02 - Compatibilidade
O site deve funcionar em navegadores modernos (Chrome, Firefox, Edge, Safari).

7. Considerações finais:

 O projeto “Software Educacional: IDEs em Foco” foi desenvolvido conforme o planejamento proposto, atendendo aos requisitos funcionais e não funcionais definidos. Assim cumprindo seu objetivo principal de disponibilizar uma ferramenta informativa voltada ao ambiente educacional, auxiliando estudantes ou iniciantes em programação na escolha de qual ferramenta é a mais adequada às suas necessidades.
De forma geral, o projeto alcançou os resultados esperados e contribuiu para o aprimoramento técnico dos participantes, uma vez que possibilitou a utilização de conhecimentos de programação web, estruturação de conteúdo e design responsivo. Além disso, foi cumprido o objetivo de representar uma ferramenta útil para fins didáticos e de apoio ao aprendizado em cursos da área de tecnologia.
